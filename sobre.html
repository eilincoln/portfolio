Minha jornada na tecnologia começou na área de infraestrutura e suporte, onde atuo como Responsável Técnico de TI. Lidar diariamente com servidores, redes e a consistência de bancos de dados PostgreSQL me deu uma disciplina analítica rígida: aprendi a depurar erros com método, entender a arquitetura dos sistemas e focar na resolução definitiva de problemas.

Essa mentalidade lógica me atraiu naturalmente para o Desenvolvimento Front-end. Criar interfaces não é apenas alinhar elementos visualmente, mas traduzir regras de negócio complexas em código limpo, semântico e performático.

🧱 Stack Técnico Atual

Atualmente, divido minha rotina de estudos e aplicação prática nos seguintes pilares estruturais:

  • Desenvolvimento Estrutural: HTML5 semântico, acessibilidade web e otimizações de SEO.
  • Estilização Avançada: CSS3 modular, variáveis nativas, arquitetura Flexbox e layouts matemáticos com CSS Grid.
  • Lógica e Dados: Fundamentos de JavaScript aplicados à interatividade e manipulação do ecossistema de dados.
  • Ferramental Corporativo: Controle de versão profissional com Git e gerenciamento de repositórios no GitHub.

Busco constantemente o equilíbrio exato entre a engenharia de software organizada e o design intuitivo. Meu objetivo é construir soluções digitais completas que agreguem valor real a empresas, automatizem processos e ofereçam experiências digitais impecáveis.
